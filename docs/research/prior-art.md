# Prior-art research

The implementation was informed by public Agent Skills and standards covering browser testing, accessibility, web quality, UX writing, design heuristics, anti-slop patterns, iOS accessibility, and security verification.

Principles adopted include progressive disclosure, evidence preservation, reconnaissance before interaction, deterministic helper scripts, component acceptance criteria, explicit manual-review queues, complete-process accessibility, project documentation as context, and specialist orchestration.

The suite independently implements its workflow and reporting contract. See `docs/licensing/ledger.md` for reviewed projects and licences.
