# Methodology

Inventory routes, roles, workflows, components, and important states before sampling. Test complete user processes, not only isolated pages. Combine automated tools with manual keyboard, focus, reflow, and assistive-technology checks. A zero-violation automated result is never a compliance claim.

Prioritise primary journeys, authentication, forms, account management, transactional steps, dialogs, dynamic updates, and shared components. Record every untested criterion and environment limitation.
