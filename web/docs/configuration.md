# Configuration

Sills first reads the project's own documentation and conventions. Optional content configuration may define purpose, audiences, voice, reading level, terminology, locales, and product claims.

The default output is `audit/YYYY-MM-DD/`. A user may provide another path. The audit skill may ask only when an existing convention conflicts or the request requires credentials, roles, or environment information that cannot be discovered safely.
