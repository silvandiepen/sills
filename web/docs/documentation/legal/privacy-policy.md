# Privacy policy

Last updated: July 15, 2026

This policy explains how Sills handles information in connection with the public website, documentation, GitHub repository, npm packages, and related project infrastructure.

## Who provides Sills

Sills is an open source project maintained by its contributors. The source code is published at [github.com/silvandiepen/sills](https://github.com/silvandiepen/sills).

## Information the website processes

The Sills website is hosted on Cloudflare Pages. When you visit the website, Cloudflare may process request information such as IP address, browser details, requested URL, timestamps, and security events in order to serve the site, protect it, and operate its network.

The Sills website does not intentionally create user accounts, ask for payment details, or collect form submissions.

## GitHub, npm, and third-party services

If you interact with Sills through GitHub, npm, Cloudflare, or another third-party service, that service may process information according to its own policies. Examples include opening an issue, starring the repository, installing a package, downloading package metadata, or viewing raw files from GitHub.

Sills does not control those third-party services.

## Package behavior

Sills packages are designed to run locally in your development environment. The Sills packages do not intentionally send telemetry to the Sills maintainers.

Some commands may invoke tools, package registries, browsers, GitHub, npm, or other services depending on how you run them. Those services may process their own logs or request data.

## Audit output

Sills audits may inspect source code, documentation, running applications, screenshots, terminal output, logs, accessibility scan results, package metadata, and other evidence available in the audited project or selected runtime environment.

Audit output is written to the selected audit output directory. You are responsible for deciding what audit output can be committed, shared, uploaded, or published. Do not publish audit reports that contain secrets, personal data, private customer data, proprietary source code, or confidential operational details unless you have reviewed and approved that disclosure.

## Cookies and analytics

The Sills website does not intentionally set project analytics cookies. Cloudflare or other infrastructure providers may use technical mechanisms needed to deliver, protect, and measure their services.

## Contact

For privacy questions about Sills itself, open an issue in the GitHub repository. Do not include secrets or private personal data in public issues.

For privacy questions about GitHub, npm, Cloudflare, or another third-party service, contact that service directly.
